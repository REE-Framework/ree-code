# Scientific scope of v0.1.0

## What this release contains

This release contains executable demonstrations of finite-speed propagation,
finite-resolution readout, conservative boundary accounting, selected registered
formulae, and reproducibility support.

## What this release does not contain

It does not contain:

- a complete cosmological parameter-inference pipeline;
- detector-calibrated P1–P9 analyses;
- a full black-hole or horizon simulation;
- a derivation of the REE ontology from the code;
- evidence that any registered discriminator has passed;
- a post-quantum implementation of the full REE computational requirement.

## Interpretation rule

A code path is not canonical merely because it is executable. Each module states
whether it is registered, validating, demonstrative, exploratory, or metadata
only. Physical promotion remains controlled by the manuscript and immutable
research record.
