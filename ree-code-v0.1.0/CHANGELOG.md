# Changelog

All notable changes to the public code package are recorded here.

## [0.1.0] — 2026-07-16

### Added

- Initial installable `ree-code` Python distribution.
- Finite-speed damped-wave front demonstration with CFL validation.
- Finite-resolution readout and RRIP-style comparison utilities.
- Conservative one-dimensional boundary-ledger demonstration.
- Versioned discriminator metadata and P7 frequency-scaling helper.
- Phase-resolution and stacking calculations.
- Reproducibility manifest utilities.
- Command-line interface, examples, tests, documentation, and CI workflow.

### Scientific scope

This release is an initial demonstration and validation package. It does not
claim to implement the complete REE physical programme or all registered
analysis pipelines.
